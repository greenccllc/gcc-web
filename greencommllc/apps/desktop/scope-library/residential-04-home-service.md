# Scope — Home Service (Residential)

**Applies to:** One-off service calls, WiFi surveys, and post-install walkthroughs. Smaller than a full install package.

---

## Inclusions

### WiFi site survey

- Walk-the-house signal-strength measurement using a commercial-grade site-survey tool
- Heat-map generated showing coverage gaps
- Recommendation report — how many access points, where to place them, wired backhaul or mesh

### Service call

- Troubleshoot an existing network or AV problem
- Repair or replace a failed component (cable, jack, router, access point)
- Up to 2 hours labor included in the standard service-call rate; additional labor T&M at service rate
- No return fee if a problem reappears within 30 days of the same service

### Post-install walkthrough

- 30 to 60-minute session with the homeowner
- Overview of what was installed
- App setup on the primary phone or tablet
- Written one-pager — what's where, credentials, support contact

## Deliverables

- Service-call work order with before/after notes
- WiFi survey PDF report with heat map
- Walkthrough one-pager

## Exclusions

- Repair of equipment not installed by GCC (available, but no warranty on prior work)
- Computer or IoT device setup beyond network connectivity (a separate "IT consulting" engagement)
- ISP troubleshooting beyond the router

## Service-call rates (reference for quoting)

- Standard residential service call: $150 flat + $95/hour beyond 2 hours
- After-hours (after 5 PM weekdays, weekends): 1.5× multiplier — never a premium add on a full install; only applies to service calls specifically booked outside standard hours

## Rule of thumb

If the call reveals a fundamental design problem (not enough drops, no structured-media panel, coverage gaps too large to patch with one more AP), convert the conversation into a **Home Network scope** quote rather than chaining service calls.
