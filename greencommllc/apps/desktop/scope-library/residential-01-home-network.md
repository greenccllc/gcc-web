# Scope — Home Network (Residential)

**Applies to:** Single-family homeowner proposals. Smart-home, remote-work-from-home, or whole-home WiFi.

---

## Inclusions

- Cat6 or Cat6A UTP drops from a central structured-media panel to each planned location — TV locations, office desks, smart-home hubs, camera locations, wireless access-point ceiling locations
- Structured-media panel install — 30" or 42" recessed can, located in a utility room, basement, or closet. Includes patch panel and mounting of router, switch, and modem
- WiFi mesh or ceiling-mount access-point placement — walk-the-house survey to confirm coverage; typical 2,500–5,000 sqft home uses 2–3 access points
- Router configuration — SSID and password setup, basic guest network, device priorities (Quality of Service for video calls and streaming)
- Patch-cord labeling and cable dress at the panel
- Cable re-run or patch repair for existing drops that show signs of damage or interference

## Deliverables

- One-page network map — which room is on which port
- Test results — signal-strength readings at every access point, speed test at a representative sample of drops
- Router / access-point login credentials document, delivered in a sealed envelope

## Exclusions

- Internet service procurement (Spectrum, AT&T, T-Mobile home internet) — homeowner places the order, GCC times work to match install date
- High-TV-count distribution over coax (see Home AV scope)
- Whole-home audio (see Home AV scope)
- Commercial-grade rack and rack dress

## Typical package (for density-based quoting)

| Home size | Data drops | Access points |
|---|---:|---:|
| < 2,000 sqft | 6–8 | 1–2 |
| 2,000–3,500 sqft | 10–14 | 2–3 |
| 3,500–6,000 sqft | 16–22 | 3–4 |
| 6,000+ sqft | 24+ | 4+ |

## Plain-English substitutions (for homeowner voice)

- "Cat6A plenum" → "a future-proof cable grade that handles gigabit speeds today and the generation after next"
- "Structured-media panel" → "the small network closet that organizes every cable in one place"
- "Access points" → "the ceiling-mounted WiFi transmitters, spaced through the house so the signal is strong in every room"
