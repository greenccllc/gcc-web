# Scope — Certification & Documentation (Commercial)

**Applies to:** Every commercial project — this scope is never optional and never value-engineered out.

---

## Inclusions

### Copper certification

- **Fluke DSX-8000 CableAnalyzer** — owned in-house, not rented
- Every Cat6A permanent link tested to Class EA (TIA/EIA-568-D)
- Results saved as .flw files, exported to **LinkWare PDF** with per-port detail: length, NEXT, FEXT, return loss, insertion loss, propagation delay, delay skew, resistance, map diagram
- PDF bundled by rack or IDF, delivered at closeout

### Fiber certification

- **OTDR** single-mode at 1310nm / 1550nm, multi-mode at 850nm / 1300nm
- Trace files (.sor) per strand, per direction where applicable
- **LSPM** end-to-end insertion loss measurement with matched jumpers
- Loss budget calculation documented per fiber run

### Coax certification (where in scope)

- Sweep test 5 MHz – 1 GHz per RG6 run
- Attenuation and return-loss pass/fail documented

### TIA-606-C labeling

- Class 3 administration (every port, every cable end, every rack, every work-area outlet)
- Brady M611 printed labels, thermal transfer, 90°F–sustained adhesive
- Label schedule delivered at closeout — CSV and PDF

### As-built documentation

- Marked-up drawings returned from the field, digitized into PDF
- CAD as-builts on request (billed T&M at CAD rate unless included in base scope)
- Rack elevations — one per rack
- One-line diagrams for fiber backbone and critical security runs

## Deliverables at closeout

| Document | Format |
|---|---|
| LinkWare PDF certification | PDF (one per IDF / rack) |
| OTDR trace files | .sor (one per strand) |
| LSPM loss report | PDF |
| TIA-606-C label schedule | PDF + CSV |
| As-built drawings | PDF (CAD optional) |
| Rack elevations | PDF |
| Product data sheets / MSDS index | PDF |
| Warranty registration | PDF |
| Warranty certificate (lifetime workmanship) | PDF |

## Exclusions

- Ongoing periodic re-certification (available under separate Managed Services agreement)
- Certification of cables not installed by GCC (available as a separate service call, T&M)

## Standards

TIA/EIA-568-D Class EA · TIA-606-C · IEC 61280 (fiber testing) · SCTE 40 (coax)
