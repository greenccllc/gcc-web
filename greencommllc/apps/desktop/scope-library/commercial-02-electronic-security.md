# Scope — Electronic Security (Commercial)

**Division:** 28 13 00 (Access Control) · 28 23 00 (Video Surveillance) · 28 31 00 (Intrusion Detection)
**Applies to:** Commercial bids with cameras, access control, intrusion, or intercom in scope.

---

## Inclusions — Video Surveillance

- IP camera installation — bullet, dome, fisheye, or PTZ per design
- Pre-assembly and bench-test of each camera prior to field install
- Mounting hardware, weather shrouds, and vandal covers per manufacturer specification
- VMS / NVR integration — camera discovery, basic view-grouping, permissions handed off to IT
- 4MP / 8MP resolution assumed unless specified
- Preferred manufacturers: **Avigilon** (enterprise), Ubiquiti UniFi Protect (small/mid)

## Inclusions — Access Control

- Card readers — proximity or smart-card per design (HID iClass SE assumed unless specified)
- Electric strikes, magnetic locks (mag locks), or electrified mortise per door schedule
- Door position switches (DPS) and request-to-exit (REX) sensors
- Head-end wiring to the access-control panel; low-voltage power supplies with battery backup
- Preferred control platforms: **Mercury**, HID Signo / Edge

## Inclusions — Intrusion Detection

- Motion detectors, door/window contacts, glass-break sensors
- Keypads at entry points
- Intrusion panel wiring and commissioning
- Preferred manufacturer: **Bosch**

## Inclusions — Intercom

- Entry-station intercom with camera (door station)
- Interior master-station install and wiring
- Integration with VMS or standalone operation

## Deliverables

- Camera aim sheet — coverage diagram per camera
- Door schedule with hardware part numbers and panel addresses
- Programming sheet for access-control users (blank — to be completed by IT)
- Warranty registration and 5-year GCC-furnished hardware warranty

## Exclusions

- Head-end server / NVR hardware beyond what is specified (GCC will mount, rack, and network-connect — VMS software licensing is client-procured)
- Card / fob inventory beyond a starter quantity of 25
- Ongoing monitoring service (central station)
- Fire-alarm integration or relay to FACP (out of scope — see Hard Rules)
- Public-address / mass-notification beyond standard paging speakers

## Standards

UL 294 (access control) · UL 2050 (alarm) · NEC Article 725 · ADA reach-range compliance for readers and keypads
