# Milestone Schedule Templates

Pulled into the Schedule section of proposals. Durations are indicative — adjust based on actual scope.

---

## Template 1: K-12 Summer Work (school-year windows)

Best fit: district-wide classroom cabling, drop adds, camera refresh, access-control upgrades in schools. Window is the 8–10 weeks between Memorial Day and mid-August.

| Milestone | Duration | Notes |
|---|---|---|
| Contract award + submittal package | Week 1 | Material ordered, submittals delivered to district |
| Material delivery + staging | Week 2 | Dock at district warehouse or staging site |
| Mobilization | Week 3 day 1 | Crew on site, safety meeting with district |
| Rough-in (cabling, pathway, backboxes) | Weeks 3–5 | School closed, full access |
| Trim-out (terminations, faceplates, devices) | Weeks 5–7 | |
| Certification + labeling | Week 7 | Fluke DSX-8000 every port, Brady TIA-606-C |
| Closeout package + punch walk | Week 8 | LinkWare PDF, as-builts, punch list |
| Substantial completion | Week 8 | Prior to staff return for PD / in-service days |

## Template 2: New Commercial Construction (phased with GC)

Best fit: new-construction commercial buildings, subcontract under a GC's master CPM schedule.

| Milestone | Duration | Notes |
|---|---|---|
| Award + mobilization | Contract day 1–14 | Submittals, insurance delivery, pre-construction meeting |
| Pathway rough-in | Aligned with framing | J-hooks, conduit sleeves, cable tray, firestop |
| Backboard + rack install | Aligned with drywall | IDFs, MDF, rack-in-place |
| Cable pull (rough-in) | Aligned with above-ceiling inspection | Pulled and dressed, not terminated |
| Above-ceiling + wall close-in | Aligned with GC cover-up | Final ceiling close, paint |
| Trim-out (terminations, faceplates, devices) | Aligned with trim / finish trades | |
| Certification + labeling | 2 weeks pre-SC | Fluke DSX-8000, OTDR, Brady labels |
| Owner training + closeout | 1 week pre-SC | Walkthrough with IT / facilities, closeout package |
| Substantial completion | Per GC schedule | |

## Template 3: Commercial Renovation (occupied space)

Best fit: tenant buildout, refresh, or refit in an active building — healthcare, senior living, office renovations.

| Milestone | Duration | Notes |
|---|---|---|
| Pre-construction walk + ICRA plan | Week -2 to -1 | If healthcare / senior living |
| Award + submittal package | Week 1 | Including infection-control plan |
| After-hours rough-in (Phase 1) | Weeks 2–3 | Weekend / evening hours if occupied |
| Rough-in + trim-out (Phase 2) | Weeks 4–5 | |
| Certification + labeling | Week 5 | |
| Closeout + training | Week 6 | |
| Substantial completion | Week 6 | |

## Template 4: Residential Smart-Home (new or renovation)

Best fit: single-family home, smart-home and home-network install.

| Milestone | Duration | Notes |
|---|---|---|
| Contract signing + deposit | Day 1 | 30% deposit |
| Material order + delivery | Days 2–10 | Staged at contractor's shop or homeowner's garage |
| Pre-drywall rough-in (new construction) | 2–3 days | Only applicable if home is under construction |
| Main install visit (cable pull, device mount, terminations) | 2–4 days | Consecutive days, homeowner on-site to confirm placement |
| Commissioning + walkthrough | Half-day | Router and AP setup, homeowner training |
| Final payment + walkthrough sign-off | Same day as commissioning | 30% balance due |

## Rules for schedule inclusion in proposals

- Milestones listed as **natural numbered sequences** — this is the one case where numbered lists are allowed in client-facing copy.
- Dates are never committed in the base proposal. Use "Week 1," "Week 2," etc. Dates are locked after contract signing.
- "Substantial completion" is defined as: certification complete, closeout package delivered, punch walk executed. Not: 100% punch clearance (that happens in warranty period).
- Never commit to a schedule shorter than 80% of what the estimating methodology supports. If the GC / owner requires it, that's an accelerated-schedule alternate, priced separately.
