## 03 — Scope of Work

### 3.1 Base Scope — Structured Cabling

GCC will furnish and install a new Cat6A plenum-rated structured cabling system across the three academic wings (A, B, C) and the field house annex. Installation follows TIA/EIA-568-D commercial building cabling standards and NEC Articles 725 and 800.

Inclusions:

- 420 Cat6A plenum data drops — classroom, office, AP coverage, WAP, and POS locations
- All drops home-run to the second-floor IDF via existing J-hook pathway, supplemented where required
- Termination on new Leviton Atlas-X1 24-port patch panels in the IDF
- Room-end termination on Leviton QuickPort Cat6A keystones in single-gang decora plates
- Full Fluke DSX-8000 Cat6A permanent-link certification for every drop
- TIA-606-C labeling: every cable, every jack, every patch-panel port
- LinkWare PDF test report delivered as part of closeout

### 3.2 Base Scope — Electronic Security

GCC will furnish and install an IP video surveillance expansion covering interior and exterior coverage per the architect's security diagram dated 2026-03-14.

Inclusions:

- 47 IP camera installations — mix of interior dome, exterior bullet, and parking-lot vari-focal
- Hikvision DS-2CD series cameras per spec (OFCI — GCC installs only)
- Cat6A PoE++ homeruns to the new Ubiquiti UniFi Pro Max 48 PoE aggregation switch
- Cable tray and J-hook pathway supplement where the existing infrastructure is undersized
- NVR integration — GCC coordinates with District IT; NVR is OFCI
- 18 access-controlled door positions — reader, REX, door contact, strike tie-in — terminating at a new Mercury Security LP1502 board in the IDF

### 3.3 Base Scope — Telecom Room

Inclusions:

- One new 42U free-standing rack, seismic-bolted, in the second-floor IDF
- Ladder rack from the rack to the existing cable-tray pathway
- New 48-port Ubiquiti UniFi Pro Max 48 PoE aggregation switch (furnished by GCC, wired by GCC)
- Copper patch field for the 420 data drops
- Power strip (PDU) — 20A, 16-outlet, surge-protected
- TIA-606-C labeling throughout the rack
- Rack elevation drawing delivered as part of closeout
