# Scope — Home Security (Residential)

**Applies to:** Homeowner smart-home security — doorbell cameras, smart locks, exterior cameras.

---

## Inclusions

- Doorbell camera — wired hardwire to existing door-chime transformer, or battery-powered if no wired chime
- Smart locks — deadbolt replacement, WiFi or Z-Wave connected, app-based unlock and temporary codes
- Exterior cameras — WiFi or PoE, installed under eaves or at soffits for weather protection
- WiFi / app-based access integration — user accounts, shared access for family, temporary codes for guests or service providers
- Notification setup — phone alerts for motion, door open/close, package detection if camera supports
- Homeowner walkthrough — 30-minute session on the app, recording storage, and privacy settings

## Deliverables

- Camera placement diagram
- App setup guide personalized for the household
- Warranty registration for all installed devices

## Exclusions

- Central-station monitoring (third-party service, homeowner-contracted)
- Alarm panel installation (intrusion detection) — available separately
- Glass-break or window-sensor retrofits beyond what the homeowner has selected

## Preferred equipment

| Category | Consumer | Prosumer |
|---|---|---|
| Doorbell cameras | Google Nest Doorbell | Ubiquiti UniFi G4 Doorbell Pro |
| Smart locks | Schlage Encode | Yale Assure 2 with module |
| Exterior cameras | Ubiquiti UniFi G4 / G5 Bullet | Avigilon H5A (rare in residential) |

GCC recommends against cloud-only systems that lose function when the internet is down — a simple UPS in the structured-media panel avoids this.

## Plain-English substitutions

- "Wired hardwire" → "using the existing doorbell wiring, so it never needs to be charged"
- "Z-Wave" → "a wireless protocol that reaches further into the house than WiFi and uses very little battery"
