# Scope — Project Management (Commercial)

**Applies to:** Every commercial project. Pulled into Section "How we deliver" or Section 02 Approach.

---

## Inclusions

### Pre-construction

- Site survey — walk the building pre-bid where possible; document ceiling type, pathway availability, IDF/MDF locations, telecom room conditions
- RCDD review of design drawings and specifications — commentary on constructability, code compliance, and any open design questions (internal only; no RFIs issued)
- ICRA risk assessment for healthcare and senior-living projects — Class 3 / Class 4 mitigation plans
- Submittals package — product data sheets, shop drawings, color samples, compliance letters
- Schedule coordination with GC — integrated into master CPM schedule

### During construction

- Dedicated project manager assigned at award — same PM start-to-finish
- Weekly progress updates with photo-free status (cabling count to date, certification status, issue log)
- Two-person minimum crew on every site visit; foreman ratio 1:3 maximum
- OSHA 10 for all crew; OSHA 30 for supervisors
- Daily safety walks, toolbox talks weekly
- Change-order process — written quote with labor + material + markup, 3-day owner approval window, never start unauthorized scope

### Closeout

- Closeout package — see Certification & Documentation scope
- Punch walk with GC or owner; punch-list clearance within 5 business days of creation
- Warranty registration and handoff meeting with IT / facilities
- Managed Services onboarding (if elected) scheduled for within 30 days post-substantial-completion

## Deliverables

- PM contact sheet (name, phone, email, escalation ladder)
- Schedule of values tied to contract milestones
- Weekly progress reports through substantial completion
- Final closeout package

## Exclusions

- GC-furnished temporary power, lighting, restrooms, dumpsters, hoists
- Permit pull beyond low-voltage permit (electrical permit, building permit by GC)
- OAC (Owner-Architect-Contractor) meeting attendance beyond what the GC schedules
- Hazardous materials abatement

## Certifications available on request

- BICSI-certified installers on site (RCDD available for review and stamping)
- ICRA Class 3 / Class 4 trained foreman for healthcare work
