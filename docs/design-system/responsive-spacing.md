# Responsive Breakpoint & Spacing Standards

## Breakpoints

Use mobile-first breakpoints:

- `xs`: 0–479px
- `sm`: 480–767px
- `md`: 768–1023px
- `lg`: 1024–1279px
- `xl`: 1280px+

## Layout Behavior by Breakpoint

- `xs/sm`: Single-column content; collapsed nav; stacked form actions.
- `md`: Two-column where appropriate; moderate gutters.
- `lg/xl`: Multi-column dashboards; wider data layouts; preserved whitespace.

## Spacing Rhythm

- Base rhythm: 4px.
- Preferred spacing scale: 4, 8, 12, 16, 24, 32, 48, 64.
- Section spacing should scale up with viewport width.

## Container & Gutter Standards

- Default page horizontal padding:
  - `xs`: 16px
  - `sm`: 16–20px
  - `md`: 24px
  - `lg+`: 32px
- Keep readable text measure (~60–80 characters per line).

## Interaction Density Rules

- Minimum touch target: 44x44px for primary interactive controls.
- Maintain clear vertical spacing between interactive elements to avoid accidental taps.
