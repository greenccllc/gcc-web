Install on the Samsung

  1. Open this page in Chrome on the phone.
  2. Tap Download APK above.
  3. Allow Chrome to install from unknown sources when prompted.
  4. Open the downloaded app-release.apk and install over any previous version.
  5. Launch GCC Phone Sync → tap Sign in with Google → it opens api.greencommllc.com/admin/phone-sync/pair in Chrome Custom Tab; your existing greencommllc.com Workspace session signs you in silently.
  6. Tap Pair this device on the admin page; the deep link bounces back into the app and connection is set.

What's in 1.3.0

  • 3-tab UI: Today (status + sync) · Health (perms + diagnostics) · Settings (advanced + sign out).
